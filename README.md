# myFirstVueProject
 第一个vue项目，完成一个小功能，一个to-do list
# 功能
添加今日待办事项和明日代办事项<br>，
### 可以删除今日代办事项和明日代办事项<br>
### 点击代办事项列表项可以更新成完成状态。
### 可以查看前一天待办事项。
# 使用的技术点（HTML5 + bootstrap + vue）：
### 简单的Vue指令如：v-bind，v-if，v-show，v-for等。
### Vue的过渡知识：transition，transition-group等。
### Bootstrap的样式。
### HTML5的本地存储localStorage
